# LOD Helmholtz Formulation

The first implementation targets the homogeneous impedance Helmholtz model
\[
-\Delta u-\kappa^2u=f\quad\text{in }\Omega,\qquad
\nabla u\cdot\nu-i\kappa u=0\quad\text{on }\partial\Omega.
\]
The sesquilinear form is
\[
a(u,v)=(\nabla u,\nabla v)_\Omega-\kappa^2(u,v)_\Omega-i\kappa(u,v)_{\partial\Omega}.
\]

The fine reference matrix is
```text
A = K - k^2 M - i k M_boundary.
```
The LOD trial and test spaces are generated by localized primal and adjoint correctors. For the default implementation, \(I_H\) is the existing `weightedClementP1` coarse-from-fine transfer. The local constraints are restrictions of the relevant Clement rows to each patch's free local fine nodes.

The first 2D reproduction regime is small, controlled, and matrix-based:
- nested P1 triangular meshes from `squaremesh`;
- all boundary edges treated as Robin/impedance edges;
- \(H\kappa\lesssim 1\);
- \(\ell\) increased with wave number or decreasing \(H\);
- errors measured in the \(\kappa\)-weighted energy norm.

The 3D completion uses the same constrained saddle-corrector algebra on nested P1 tetrahedral meshes from `cubemesh`. All boundary faces are impedance faces, the energy comparison uses `K + k^2 M`, and the default verification remains smoke- and mini-sweep-scale rather than a full 3D reproduction study.

The implementation intentionally does not use the older spectral/eigenfunction LOD prototype or `E:\bwzheng\Two_level_add` during development.
